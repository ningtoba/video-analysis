# Entity-Level Tracking Across Video Scenes — Research Report

**Date:** 2026-06-26
**Context:** video-analysis v0.15.0 — PySceneDetect + YOLO + OpenCLIP + ChromaDB + Gradio 6
**Goal:** Assign and preserve persistent IDs to people/objects detected across multiple scenes.

---

## 1. Current State

The pipeline currently runs per-frame YOLO object detection on keyframes with **no tracking**:

- No track IDs assigned
- No cross-frame association (every frame is independent)
- No cross-scene ReID (when a person leaves and re-enters)
- Uses yolo26x.pt (~2 GB VRAM in fp16)

---

## 2. Tracker Options for Per-Frame ID Assignment

### 2.1 Built-In: Ultralytics YOLO Trackers (Recommended — Fastest Path)

Ultralytics ships **six trackers** natively. Zero additional dependencies beyond ultralytics.

| Tracker | Config | Motion Model | Camera Comp | ReID | Occlusion | License |
|---------|--------|-------------|-------------|------|-----------|---------|
| **BoT-SORT** (default) | botsort.yaml | Linear Kalman | yes sparseOptFlow/ECC | Optional | Track buffer + ReID rebinding | AGPL-3.0 |
| **ByteTrack** | bytetrack.yaml | Linear Kalman | No | No | Two-stage low-conf rescue | MIT |
| **OC-SORT** | ocsort.yaml | Obs-centric Kalman | No | No | ORU/OCM/OCR re-update | MIT |
| **Deep OC-SORT** | deepocsort.yaml | Obs-centric Kalman | Optional | Optional | OC-SORT + adaptive EMA | MIT |
| **FastTracker** | fasttrack.yaml | Kalman + rollback | No | No | Kalman rollback + bbox enlargement | AGPL-3.0 |
| **TrackTrack** | tracktrack.yaml | Kalman (NSA) | yes | Optional (HMIoU) | Iterative multi-cue + TAI | AGPL-3.0 |

**Integration pattern (modify _detect_objects_on_frames):**

```python
# Track across ALL frames (not just keyframes)
results = self._yolo_model.track(
    str(video_path),
    persist=True,  # Maintain IDs across frames
    tracker="bytetrack.yaml",  # or botsort.yaml, ocsort.yaml
    conf=self.config.yolo_confidence,
    verbose=False,
)
for r in results:
    for box in r.boxes:
        track_id = int(box.id[0]) if box.id is not None else None
        # track_id persists until object leaves frame
```

**VRAM:** ~1.5-2 GB for YOLO tracking, same as current detection.

**Limitation:** Track IDs reset at scene boundaries. Cross-scene persistence needs ReID/face recognition.

## 3. Cross-Scene Person Re-Identification (ReID)

### 3.1 Appearance-Based ReID (Deep Metric Learning)

| Model | Params | Embed Dim | VRAM | License | Notes |
|-------|--------|-----------|------|---------|-------|
| **OSNet** (OSNet-x1.0) | 3.4M | 512 | ~0.4 GB | MIT | Lightweight; 256x128 input |
| **OSNet-AIN** | 3.4M | 512 | ~0.4 GB | MIT | Cross-domain generalization |
| **ResNet50-ibn-a** | 25.6M | 2048 | ~1.0 GB | MIT | Higher accuracy |
| **FastReID** (R50-S50) | 27M | 2048 | ~1.2 GB | Apache-2.0 | Training pipeline included |
| **DINOv2 (small)** | 22M | 384 | ~0.8 GB | Apache-2.0 | General visual features |
| **CLIP ViT-B/32** | 86M | 512 | ~1.0 GB | MIT | Already in pipeline |

**Library: torchreid (deep-person-reid)** — 4.9k stars, MIT license
- Model zoo: OSNet, ResNet, DenseNet, HACNN, PCB, MLFN
- Pretrained: huggingface.co/kaiyangzhou/osnet
- ONNX/OpenVINO/TFLite export supported

**Integration:**

```python
from torchreid.utils import FeatureExtractor

extractor = FeatureExtractor(
    model_name="osnet_x1_0",
    device="cuda",
)

# For each person detection, crop and extract embedding
scene_embeddings = {}  # {scene_id: {track_id: embedding}}
for scene in scenes:
    for frame in scene.key_frames:
        for obj in frame.objects:
            if obj["label"] == "person":
                cropped = crop_image(frame.filepath, obj["bbox"])
                emb = extractor(cropped).numpy().flatten()  # 512-dim
                scene_embeddings[scene.scene_id][obj["track_id"]] = emb
```

```python
from sklearn.cluster import DBSCAN
from sklearn.preprocessing import normalize

all_embs = [(sid, tid, emb) for sid, tid, emb in collect_face_embeddings(scenes)]
emb_matrix = normalize(np.array([e[2] for e in all_embs]))
labels = DBSCAN(eps=0.5, min_samples=1, metric="cosine").fit_predict(emb_matrix)
# Each label is a unique person identity
```

## 4. Recommended Architecture

```
1. PySceneDetect -> scene_boundaries
2. Full-video YOLO tracking -> (frame_idx, bbox, track_id, class)
3. Group tracks by scene
4. Extract ReID embeddings for person tracks (OSNet or InsightFace)
5. DBSCAN clustering across all scenes -> global_entity_id
6. Store in VideoIndex / ChromaDB
```

**Key Decision:** Run tracking on ALL frames, not just keyframes.

Option A (recommended): Two-phase — track full video, extract results at keyframe timestamps
Option B: Track on keyframes only (requires dense keyframes >= 2 fps)
